# Cab Booking SQL Project

## Project Overview
This project represents a Cab Booking System designed using SQL. 
It simulates how cab booking applications store and manage data related to customers, drivers, bookings, trips, and payments.

## Objectives
- Design a relational database for a cab booking system
- Perform data analysis using SQL queries
- Extract insights such as trip details, fare calculation, and booking history

## Database Tables
- Customers
- Drivers
- Cabs
- Bookings
- Trips
- Payments

## Key SQL Concepts Used
- CREATE & INSERT statements
- JOINS (INNER, LEFT)
- GROUP BY & HAVING
- Subqueries
- Aggregate functions (SUM, COUNT, AVG)

## Tools Used
- MySQL
- GitHub

## Use Cases
- View customer booking history
- Calculate total fare per trip
- Analyze most active drivers
- Identify peak booking times
